# Responsive Freelancer Portfolio Website

## Overview

A responsive portfolio website built using HTML and CSS.

This is a responsive freelancer portfolio website that I built using only HTML and CSS. I created this project to practice building modern website layouts and improve my understanding of responsive web design. Throughout the project, I focused on writing clean code and making the website work well on different screen sizes.
